# AI Obsidian Todo Bar

macOS menu bar app that reads tasks from an **Obsidian vault** (`Inbox/tasks/`) and sends **AI-powered supportive notifications** via **OpenRouter**.

- **Menu bar icon** → macOS-native popover with task list
- **AI notifications** — generated by LLM from editable `_prompt.md` template
- **Mark done** from tray — syncs `- [ ]` → `- [x]` back to `.md` files
- **Notification history** saved to `Inbox/tasks/history-{date}.md`
- **Desktop widget** via WidgetKit (macOS 14+)

## Quick Start

1. Open `ObsidianTodoBar.xcodeproj` in Xcode
2. Build & run (macOS 14+)
3. Configure vault path and OpenRouter API key in Settings
4. Edit `Inbox/tasks/_prompt.md` to customize AI behavior

## Vault Layout

Folder structure defined in `.env`:

```
$TASKS_FOLDER/                 ← folder monitored by the app
    $PROMPT_FILE               ← AI prompt template
    history-{date}.md          ← auto-generated notification log (pattern: $HISTORY_FILE_PATTERN)
    any-task-file.md           ← tasks with YAML frontmatter
```

## Configuration

Set vault paths in `.env` at project root. App settings available via tray menu → Settings:
- Vault path (defaults to `$OBSIDIAN_VAULT_PATH`)
- OpenRouter API key
- AI model (supports any OpenRouter model ID, defaults to `$DEFAULT_AI_MODEL`)
- Check interval (defaults to `$CHECK_INTERVAL`)
- Default notification hour
