# AI Obsidian Todo Bar

macOS menu bar app for AI-powered task notifications from an Obsidian vault.

- **Menu bar** вЂ” left-click opens popover with task list, right-click for menu
- **AI notifications** вЂ” generated by LLM via OpenRouter from editable `_prompt.md` template
- **Add / Edit tasks** вЂ” native macOS window with title, date, time, recurrence, checklist
- **Recurrence** вЂ” daily, weekdays (custom selection), weekly, monthly
- **Quick actions** вЂ” per-task menu: postpone 1h, skip today, edit, delete
- **Temporary overrides** вЂ” `overrideTime` and `skipDate` auto-clear on next occurrence
- **Mark done** from tray вЂ” syncs `- [ ]` в†’ `- [x]` back to `.md` files
- **Notification history** saved to `history-{date}.md`

## Quick Start

```bash
cp .env.example .env   # edit with your paths and API key
make launch            # builds, bundles, and launches
```

## Task Format

Files in `Inbox/tasks/` use YAML frontmatter with Obsidian-compatible checkboxes:

```yaml
---
title: Task title
due: 2026-07-01
time: 14:30
recurring: daysOfWeek
days: mon,wed,fri
overrideTime: 16:00
skipDate: 2026-07-01
---

- [ ] Do the thing
```

- `due` вЂ” date for notification
- `time` вЂ” optional; defaults to 09:00
- `recurring` вЂ” `daily`, `daysOfWeek`, `weekly`, `monthly`
- `days` вЂ” only when `recurring: daysOfWeek`: comma-separated 3-letter day names (`mon,tue,wed,thu,fri,sat,sun`)
- `overrideTime` вЂ” temporary time override (auto-cleared on next occurrence)
- `skipDate` вЂ” skip task for this date (auto-cleared on next occurrence)
